============================
stack SDKとは
============================

stack SDKはstack クライアントを開発するための機能を実装したnpmパッケージです。
主に、stack APIを利用するための複雑なアカウントデータ作成処理、
及びstack APIを利用したデータのアップロード、
ダウンロード等を最小限のソースコードで実装することが可能です。


導入
---------------------------
stack SDKはnpmを利用してインストールすることが可能です。

::

$npm install stack-SDK
